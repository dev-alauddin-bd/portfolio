# Modern Full-Stack Developer Portfolio

A premium, high-performance developer portfolio built with **Next.js 15 (App Router)**, **Tailwind CSS**, and **Framer Motion**. Featuring a radical high-contrast design, multiple color themes, and human-centric UI/UX.

## 🚀 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [AOS](https://michalsnik.github.io/aos/)
- **Theme Management**: [next-themes](https://github.com/pacocoursey/next-themes)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Forms**: [EmailJS](https://www.emailjs.com/)
- **Smooth Scroll**: [Lenis](https://github.com/darkroomengineering/lenis)

## ✨ Key Features

- **Radical & Bold UI**: High-contrast, typography-focused design for maximum impact.
- **Dynamic Themes**: Support for Neon Blue, Electric Rose, Solar Orange, and Cyber Green themes.
- **Human-Centric UX**: Carefully crafted copy focused on recruiter value and user success.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.
- **Project Gallery**: Immersive showcase for featured projects with detailed case studies.
- **Skill Ecosystem**: Interactive, searchable toolkit visualization.

## 🛠️ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/dev-alauddin-bd/portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   pnpm install
   ```

3. **Set up environment variables:**
   Create a `.env` file in the root directory and add your EmailJS credentials:
   ```env
   NEXT_PUBLIC_EMAILJS_SERVICE_ID=your_service_id
   NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=your_template_id
   NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=your_public_key
   ```

4. **Run the development server:**
   ```bash
   pnpm dev
   ```
